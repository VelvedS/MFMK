# MA Food for MA Kids — static site

Deploy to Vercel:
1. Unzip this folder.
2. Drag the folder into vercel.com/new (or run: vercel). Static site — no build step.

Pages:
- index.html ............ Homepage (V4 Magazine)
- Home-V2.html .......... Alternate homepage (V5 Immersive)
- grant-information.html
- policy-priorities.html
- our-coalition.html
- faq.html
- take-action.html
- join-the-coalition.html
- share-with-your-networks.html

Fonts: Anton + Source Sans 3 are self-hosted in assets/fonts via fonts.css (no internet needed at view time).
Assets live in /assets (logos, heroes, image-slot.js) and /assets/pages (section photos).
